# 📊 People Analytics Dashboard

Dashboard interativo desenvolvido com **Python, Pandas, Plotly e Streamlit** para análise de indicadores de Recursos Humanos (People Analytics).

## 🎯 Objetivo

O objetivo deste projeto é demonstrar a construção de um dashboard analítico para visualização de métricas de RH, permitindo explorar dados de funcionários de forma simples e interativa.

## 🚀 Tecnologias Utilizadas

* Python
* Pandas
* Streamlit
* Plotly

## 📈 Funcionalidades

* Visualização da base de funcionários
* Indicadores (KPIs) de RH

  * Total de funcionários
  * Salário médio
  * Quantidade de departamentos
* Filtro por departamento
* Gráfico de funcionários por departamento
* Distribuição salarial por departamento
* Salário médio por departamento
* Insights automáticos

## ▶️ Como Executar

### 1. Clonar o repositório

```bash
git clone https://github.com/giorgeleao/people-analytics-dashboard.git
```

### 2. Entrar na pasta

```bash
cd people-analytics-dashboard
```

### 3. Instalar dependências

```bash
pip install -r requirements.txt
```

### 4. Executar aplicação

```bash
python -m streamlit run app.py
```

### 5. Abrir no navegador

```text
http://localhost:8501
```
